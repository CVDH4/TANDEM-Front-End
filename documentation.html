Documentation

Best Practices:

  • Files should be saved in a single folder.
  • Filenames should mirrors the hierarhcal structure of your object.

    i.e - MotherGoose_Page1.png, MotherGoose_Page2.png

  • Each page of a print object should be saved as a single file.
  • Files should be saved as .jpg or .png for the best results.
  • OCR is no 100% correct 100% of the time. Plan for some deviation.

Technical Specs

TANDEM is built on a Django web framework. It uses Python 2.7.9 as well as Python packages like OpenCV, PyTesseract, and NLTK to generate data.

TANDEM is a web-based image and text processer for images that contain text.
